UnityToolbox.UI.Settings.ResetButton Class Reference

A script which is placed on a UI element and acts as a button. It will reset key controls to the last know setting. It requires a SettingsManager, as well information about the ActionName and the Control (e.g. "Character Move" & "left"). The control can be left empty if no information is necessary.
