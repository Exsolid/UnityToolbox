UnityToolbox.GameplayFeatures.AI.Boids.Boid2DBase Class Referenceabstract

An abstract boid implementation which listens to the three rules cohesion, separation and alignment. Additionally, object attraction and avoidance can be set. When implemented, additional checks can be added. Requires ColliderInfo to work.
