 ►NItemsThe main namespace for item usage. All items are defined via the Item Manager Window found under the "UnityToolbox" dropdown. To use custome made items, create a new class inheriting ItemDefinition. The definition can now be selected within the window. All non static, public int, float, string, bool will define editable fields. Using the ItemDefinition as a variable in a seperate MonoBehaviour will present a selection field for all defined items. At runtime, the ItemManager is able to create item instances
