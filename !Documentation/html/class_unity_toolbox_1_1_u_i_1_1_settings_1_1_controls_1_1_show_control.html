UnityToolbox.UI.Settings.Controls.ShowControl Class Reference

This script displays the current setting for a control. It requires a SettingsManager, as well information about the ActionName and the Control (e.g. "Character Move" & "left"). The control can be left empty if no information is necessary. (e.g. "Character Jump" & "")
