UnityToolbox.UI.Dialog.DialogManager Class Reference

The DialogManager is a Module which manages the existing dialogs saved by the dialog graph. It requires UIEventManager and DisplayDialog to display the current dialog node on a canavas. Requires SaveGameManager if game states are used for the dialogs.
