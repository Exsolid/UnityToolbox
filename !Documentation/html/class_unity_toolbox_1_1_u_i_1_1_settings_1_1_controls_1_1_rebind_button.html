UnityToolbox.UI.Settings.Controls.RebindButton Class Reference

A script which is placed on a UI element and acts as a button. It will rebind key controls. It requires the UIEventManager and the SettingsManager, as well information about the ActionName and the Control (e.g. "Character Move" & "left"). The control can be left empty if no information is necessary. (e.g. "Character Jump" & "")
