UnityToolbox.GameplayFeatures.AI.Boids.Boid2DSimple Class Reference

A simple boid implementation which listens to the three rules cohesion, separation and alignment. Additionally, object attraction and avoidance can be set. It does not add any additional checks. Requires ColliderInfo to work.
