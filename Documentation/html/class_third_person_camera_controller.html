ThirdPersonCameraController Class Reference

A camera controller which lets the camera follow the player, move it and react on collision. To understand how it is set up, the the summeries of the position and anchor transform as well as the camera. Requires the PlayerPrefKeys and SettingsManager.

Private Attributes

Transform _targetToFollow
 The target the camera should follow.
 
Transform _anchorTransform
 Should be set up as the parent transform of the _positionTransform with no parent of itself. This handels the following of the target.
 
Transform _positionTransform
 Should have the _anchorTransform as its parent and its position and rotation set to all zero. The _camera should be its child. This handels the updated position in case of a collision.
 
Camera _camera
 This needs to be a child of the _positionTransform and can be placed anywhere. The perspective set up will be the defined maximum distance to the _targetToFollow.
